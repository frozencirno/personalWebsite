My adventure into programming was a weird one, but it was fun! I first learnt Python at my secondary school, where I learnt basic programming fundimentals and how to use libaries like pygame after some time however, I grew to be unstatisfied with the langauge, as it felt too restrictive for the things I wanted to do! After this, I addmitly entered a "Tutorial hell" going to Lua; which I used to create games on Roblox where I stoped, as I learnt of the shortcomings of the company and tools. From here I wanted to create Minecraft mods! So this lead to me learning Java, where I made some basic private mods like a double jump mod. Which made use of forge's(mod loader for Minecraft) mixins which allowed to inject code into the game. From here I gained the passion to create apps, but through means which appeared easiest, so this lead to me learning C# where I made a basic GUI calculator and basic mods for Terraria. Then finally I found C/C++, I first was learning C++ but something about C seamed more appealing to me, looking at in hindsight it was likey the emphassis on procedual code rather than OOP. This is where I started making stuff on the fly instead of being tied to Tutorials and guides on the internet, as I randomly decided to make an application which creates a bitmap image containing a colour gradient. I found resources from the Microsoft doccumentation site which helped me greatly. From here I wanted to learn about networking, so I made a simple client to client console chatting application using WinSock. This took a day to make but was incredibly statisfying to make in the end. Then I thought back on my time making the bitmap generator, and I had come up with the idea of making a perlin noise bitmap generator in C. This took some time, but with the resources avalible online made it much easier this project came to help me even further in the future, as for my Computer Science NEA coursework I wanted to make a simple voxel engine where a world of cubes were dynamicly created. This came with a decsion that I don't know if it was a bad idea to this day, but I wanted to make it using OpenGL! As many people know OpenGL is a graphics API that allows programmers to interact with the GPU in laymans terms. In other words: Hard to learn! But not impossible. In the end I was able to create the 3D voxel engine alongside the write-up for the coursework. I must say, it was fun but it was also stressful at times! I remember thinking I had missed the deadline, but I was being silly; it made me cry my eyes out! I felt defeated, but with the confidence that I had due to the additionial time gave me the willpower to get it done in the week I had. Overall it was a very fun project with its bumps, to this day I don't regret it as hedonistic as that may seem. But I also learnt so much so there was a more practical side to this project; for example I grew a lot more acustomed to memory management, code structure and code readability(I went from being someone who never comments to someone who religously comments their code!)
